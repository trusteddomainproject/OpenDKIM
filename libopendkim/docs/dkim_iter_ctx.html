DKIM_NAMETABLE_CTX

[back to index]

DESCRIPTION
Purpose A DKIM_ITER_CTX stores a context in a iteration over a table (currently support DKIM_NAMETABLE only). It is created by dkim_nametable_first() function and must be freed by the caler by using dkim_iter_free() function.
NOTES
  • This is passed as the first parameter to dkim_nametable_next().
